## üìã COMPREHENSIVE GRAPH SCHEMA (PHASE 2 REDESIGN)

### üéØ **CLEAN ARCHITECTURE GRAPH SCHEMA PRINCIPLES**

**Core Principle**: Every domain entity should be modeled as graph nodes with proper relationships, not as embedded JSON properties.

**Schema Design Rules**:
1. **Domain Entities** ‚Üí **Graph Nodes**: Each business concept gets its own node type
2. **Domain Relationships** ‚Üí **Graph Edges**: All associations become explicit relationships  
3. **Properties** ‚Üí **Node Attributes**: Only primitive data on nodes (strings, numbers, booleans, dates)
4. **No JSON Embedding**: Complex objects become separate nodes with relationships
